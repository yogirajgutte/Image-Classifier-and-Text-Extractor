1) Make sure you have installed all required modules and libraries.

2) Make sure saved model is present by name 'Model_tf' in main directory of project: "./Image Classifier/".

3) For 'Search About Image' function, 'chromedriver.exe' should be present in this directory.

4) For 'Copy Text To Clipboard' function, 'Tesseract-OCR' folder should be present in this directory. 'Tesseract-OCR' must contain 'tesseract.exe'.


5) To run the program, run 'MainMenu.py'.
